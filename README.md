# 中国足球俱乐部队徽

收集整理中国足球各级别联赛俱乐部队徽矢量图形，供学习、参考。

所有队徽/标志均包含两个版本：（01）彩色版本和（02）单色版本。

因目前国内俱乐部对球队文化建设方面的缺失，一些原始设计素材比较难收集（尤其是低级别联赛球队），所以有一些队徽/标志可能与原始设计在细节上存在细微差异。
